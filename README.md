Este é o repositório oficial do website de **Cândida Dias**, especialista em comunicação estratégica para lideranças, CEOs e fundadores. O projeto foi desenvolvido com foco em uma estética premium, performance excepcional e uma experiência de usuário fluida.

## 🚀 Tecnologias Utilizadas

O projeto utiliza o que há de mais moderno no ecossistema web:

- **Framework:** [Next.js 16](https://nextjs.org/) (App Router)
- **Biblioteca UI:** [React 19](https://react.dev/)
- **Estilização:** [Tailwind CSS 4](https://tailwindcss.com/) (Vanilla CSS + Modern Engine)
- **Linguagem:** [TypeScript](https://www.typescriptlang.org/)
- **Monitoramento:** Vercel Analytics & Speed Insights
- **Deployment:** [Vercel](https://vercel.com/)

## ✨ Funcionalidades e Seções

O site está estruturado para refletir a autoridade e o posicionamento da marca:

- **Hero Section:** Mensagem de impacto e posicionamento imediato.
- **Sobre (About):** Trajetória e diferenciais estratégicos (mais de 25 anos de experiência).
- **Serviços:** Detalhamento do processo de consultoria (Diagnóstico, Posicionamento, Estruturação, Visibilidade e Acompanhamento).
- **Pensamentos (Thoughts):** Insights e visão estratégica sobre reputação e liderança.
- **Contato Integrado:** Links dinâmicos para WhatsApp, LinkedIn, E-mail e Instagram.

## 🛠️ Como rodar localmente

Certifique-se de ter o Node.js instalado em sua máquina.

1. Clone o repositório:
```bash
git clone https://github.com/DevPBDias/candida-bio-site.git
```

2. Instale as dependências:
```bash
npm install
```

3. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

4. Abra [http://localhost:3000](http://localhost:3000) no seu navegador.

## 📐 Design & Estética

O projeto segue princípios de design moderno:
- **Tipografia:** Uso das fontes *Inter* e *New York* para um contraste elegante entre o moderno e o clássico.
- **Paleta de Cores:** Focada em tons sóbrios e profissionais que transmitem confiança e autoridade.
- **Responsividade:** Totalmente adaptado para dispositivos móveis, tablets e desktops.
